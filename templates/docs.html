登录后进入「控制台 → API密钥」创建你的密钥。密钥格式为 sk-xxxxxxxx-xxxxxxxx。

2. 一行代码接入

复制# Python - OpenAI SDK
import openai
openai.api_base = "https://api.zhongdianwangzhan.com/v1"

3. 完整示例

复制import openai

openai.api_base = "https://api.zhongdianwangzhan.com/v1"
openai.api_key = "sk-your-key-here"

response = openai.ChatCompletion.create(
    model="MiniMax-Text-01",
    messages=[
        {"role": "system", "content": "你是一个专业、有帮助的AI助手"},
        {"role": "user", "content": "你好，请介绍一下你自己"}
    ],
    temperature=0.7,
    max_tokens=1000
)

print(response.choices[0].message.content)

请求参数

参数类型必填说明
modelstring是模型标识，如 MiniMax-Text-01
messagesarray是对话消息列表，每条消息需包含 role 和 content
temperaturefloat否采样温度，范围 0-2，默认 0.7。值越大越有创造性
max_tokensinteger否最大输出 Token 数，默认受模型最大长度限制
top_pfloat否核采样概率，默认 1.0
streamboolean否是否流式输出，默认 false
stopstring/array否遇到指定字符串时停止生成
frequency_penaltyfloat否频率惩罚，范围 -2.0 到 2.0
presence_penaltyfloat否存在惩罚，范围 -2.0 到 2.0

错误码

401 Unauthorized — API Key 无效或已过期，请检查密钥是否正确。
403 Forbidden — 余额不足，请先充值后再试。
429 Rate Limited — 请求过于频繁，请降低调用频率或升级套餐。
500 Internal Error — 服务器内部错误，稍后重试或联系客服。
503 Service Unavailable — 上游服务暂时不可用，已自动切换备用线路。

流式输出（SSE）

设置 stream: true 启用 Server-Sent Events 流式输出，适合实时对话场景。

复制import openai

openai.api_base = "https://api.zhongdianwangzhan.com/v1"
openai.api_key = "sk-your-key-here"

stream = openai.ChatCompletion.create(
    model="MiniMax-Text-01",
    messages=[{"role": "user", "content": "写一首诗"}],
    stream=True
)
for chunk in stream:
    print(chunk.choices[0].delta.content, end="", flush=True)
