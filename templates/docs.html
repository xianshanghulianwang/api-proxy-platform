API 文档

API智 Hub 提供 OpenAI 兼容接口，一行代码切换，零侵入接入。

接入地址

https://your-domain/v1/chat/completions

认证方式

在请求头中使用 Authorization 字段，值为 Bearer 你的API密钥

Authorization: Bearer AK-xxxxxxxx-xxxxxxxx

请求示例（cURL）

curl https://your-domain/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 你的API密钥" \
  -d '{
    "model": "MiniMax-Text-01",
    "messages": [
      {"role": "system", "content": "你是一个有帮助的助手"},
      {"role": "user", "content": "你好，请介绍一下你自己"}
    ],
    "temperature": 0.7,
    "max_tokens": 1000
  }'

Python SDK 示例

import openai

openai.api_base = "https://your-domain/v1"
openai.api_key = "AK-xxxxxxxx-xxxxxxxx"

response = openai.ChatCompletion.create(
    model = "MiniMax-Text-01",
    messages = [{"role": "user", "content": "你好"}],
    temperature = 0.7,
    max_tokens = 1000
)

print(response.choices[0].message.content)

请求参数

参数类型必填说明
modelstring是模型名称，如 MiniMax-Text-01
messagesarray是对话消息列表
temperaturefloat否采样温度，0-2，默认0.7
max_tokensinteger否最大输出Token数
streamboolean否是否流式输出，默认false

支持模型

模型标识供应商说明
MiniMax-Text-01MiniMaxMiniMax文本模型，响应快速
deepseek-chatDeepSeekDeepSeek对话模型，高性价比
doubao-pro火山引擎豆包Pro模型，稳定性好
